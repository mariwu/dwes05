# Tarea 5 - Diseño de Interfaces Web (DIW)

Proyecto de clase desarrollado para el módulo de **Diseño de Interfaces Web** del ciclo formativo de Grado Superior en **Desarrollo de Aplicaciones Web (DAW)**.

## 📋 Descripción

Página web que muestra distintas técnicas de animación y visualización de datos mediante diagramas de barras interactivos. El proyecto está dividido en 5 ejercicios, cada uno enfocado en una tecnología diferente.

## 🚀 Ejercicios

- **CSS-TF** – Transformaciones CSS (escalado, desplazamiento, rotación)
- **CSS-TS** – Transiciones CSS aplicadas a barras de progreso
- **CSS-AN** – Animaciones CSS con control de pausa y reanudación
- **Canvas** – Diagrama de barras dibujado con HTML5 Canvas y JavaScript
- **SVG** – Diagrama de barras animado con SVG, incluyendo trayectoria de una bola interactiva

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (transformaciones, transiciones y animaciones)
- JavaScript
- Canvas API
- SVG
